## Descripcion
Este repositorio contiene los proyectos, practicas y actividades desarrolladas durante el curso de Diseño de Aplicaciones Web. El objetivo de la materia es aplicar principios de diseño, usabilidad y desarrollo para la creacion de aplicaciones web funcionales y atractivas.

- Integrar funcionalidades dinamicas en aplicaciones web.
